


<h2>Cloning the Repository</h2>

1. Open terminal/command prompt

2. Navigate to the folder where you want the project:

<code>cd path/to/your/folder</code>

3. Clone the repository using HTTPS:
   
<code>git clone https://github.com/soft-ts/HybridSchoolManagement-.git</code>

4. Enter the project folder:

<code>cd HybridSchoolManagement</code>

<h2>Contributing</h2>

1. Fork the repository

2. Create a new branch:

<code>git checkout -b feature-name</code>


3. Make your changes

4. Commit your changes:

<code>git commit -m "Add feature"</code>


5. Push to your branch:

<code>git push origin feature-name</code>


6. Open a pull request

<h2>Running a Server</h2>

1. Navigate to data folder

2. Open cmd and run 

<code>nodemon index.js</code>

<h2> Running Ionic App</h2>

1. Navigate to termProject and open command prompt

   <code> ionic serve -o</code>
